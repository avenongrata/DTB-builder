# DTB-builder
Allows you delete devices and add parameters to devices. 

It's custom solution and needs next files to work correct:
  - adi-fmcomms2.dtsi
  - ethernet.dtsi
  - mod_pl.dtsi
  - pcw.dtsi
  - pl_ad9361.dtsi
  - pl_int_rs485.dtsi
  - pl_software.dtsi
  - pl.dtsi
  - qspi.dtsi
  - zynq-7000.dtsi
  - system-top.dts
